Scripting Avanzado y Automatización

Una vez dominas los comandos individuales, es hora de combinarlos en scripts para resolver problemas complejos.

Bucles (Loops)

La estructura Foreach es vital para iterar sobre colecciones de objetos.

$servicios = Get-Service
foreach ($servicio in $servicios) {
    if ($servicio.Status -eq 'Stopped') {
        Write-Host "El servicio $($servicio.Name) está detenido."
    }
}

Crea código reutilizable definiendo tus propias funciones.

        [string]$NombrePC

Manejo de Errores

Usa Try-Catch para evitar que tus scripts fallen silenciosamente.

try {
    Get-Content archivo_inexistente.txt -ErrorAction Stop
} catch {
    Write-Warning "No se pudo leer el archivo: $_"
}
