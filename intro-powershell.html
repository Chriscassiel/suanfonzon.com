Introducción a PowerShell

PowerShell es mucho más que una simple consola de comandos. Es una potente herramienta de automatización y configuración basada en tareas.

¿Qué es exactamente?

PowerShell es una solución de automatización de tareas multiplataforma formada por un shell de línea de comandos, un lenguaje de scripting y un marco de gestión de configuración. Se ejecuta en Windows, Linux y macOS.

¿Por qué aprenderlo?

  • Automatización: Realiza tareas repetitivas en segundos.
  • Gestión del sistema: Control total sobre el sistema operativo y aplicaciones.
  • Cloud: Es esencial para administrar Azure y Microsoft 365.

Primeros pasos

Para abrir PowerShell en Windows, simplemente pulsa la tecla Windows y escribe "PowerShell". Verás una ventana azul esperando tus órdenes.

Write-Host "Hola Mundo desde PowerShell"
